How to Create a Github Webpage

1. Go to Github Website

Go to the Github Website by clicking HERE

2. Login to Github

Log into Github Website. If you don't have Github login, then create a login first.

3. Create Repository

Click "Create Repository" to create your repository

4. Select Repository Name

Your repository name should be "<>.github.io"

5. Upload Your Files

Click "upload existing files" button

6. Upload Your Files and Committ

Upload any files that you'd like to use on your webpage

7. Create a New HTML File

Clicl "Add File", then "Create New File"

8. Add your HTML code and Committ

Name the new time "index.html" and Add your HTML code, then click "Committ"

9. Go to your Github WebPage

Open a new browser tab and type in your github page address: "<>.github.io"
